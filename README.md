# phpbin
> Collaborative PHP Debugging App

Check out the demo: [phpb.in/s/75b2690c4406](http://phpb.in/s/75b2690c4406)

## Install

- Install mongodb
- Run `npm install && npm run build`

## Run

```
npm start
```
